# Global Video Game Sales Analysis Using Python

##  Project Description
This project analyzes global video game sales data using Python. 
The goal is to identify trends across genres, platforms, regions, and years 
using data visualization techniques.

##  Dataset
- Source: Kaggle – Global Video Game Sales Dataset
- Records: 16,000+ video games
- Features include sales across NA, EU, JP, and other regions

##  Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

##  Key Analysis Performed
- Data cleaning and preprocessing
- Genre-wise and platform-wise sales analysis
- Year-wise global sales trends
- Region-wise sales comparison
- Publisher performance analysis

##  Results & Insights
- Action and Sports genres dominate global sales
- North America is the largest video game market
- Sales peaked around 2008–2010

##  Conclusion
Data visualization helps uncover meaningful patterns in global video game sales
and supports data-driven decision-making.
